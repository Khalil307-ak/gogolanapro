الأساسيات وكيفية البدء مع بايثون

بايثون هي لغة برمجة قوية وسهلة التعلم تُستخدم في تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، وتعلم الآلة. في هذا الدليل، سنتناول الأساسيات التي تحتاجها لبدء رحلتك مع بايثون، بما في ذلك التثبيت، كتابة الكود، وتطبيق مشاريع عملية بسيطة.

لماذا تتعلم بايثون؟

بايثون تتميز بسهولة بناء الجملة، مما يجعلها مثالية للمبتدئين. كما أنها تحتوي على مكتبات قوية مثل pandas، numpy، وtensorflow التي تُستخدم في مجالات متقدمة. بايثون هي الخيار الأول للعديد من الشركات مثل Google وNetflix.

تثبيت بايثون

للبدء، قم بتحميل أحدث نسخة من بايثون من موقع بايثون الرسمي. اتبع الخطوات التالية:

  • لنظام ويندوز: قم بتنزيل المثبت وتأكد من تحديد خيار "إضافة بايثون إلى PATH".
  • لنظام macOS: استخدم Homebrew (brew install python) أو المثبت من الموقع الرسمي.
  • لنظام لينكس: استخدم الأمر sudo apt install python3 (لأوبونتو) أو مدير الحزم الخاص بنظامك.
  • تحقق من التثبيت باستخدام الأمر: python3 --version أو python --version.

خطAnime طوات البداية الأولى

ابدأ بكتابة برامج بسيطة لفهم بناء الجملة في بايثون:

# أول برنامج بايثون print("Hello, World!")

احفظ الكود في ملف hello.py وشغله باستخدام الأمر python3 hello.py.

أساسيات بناء الجملة

بايثون يعتمد على بساطة الكود. إليك بعض الأمثلة الأساسية:

  • المتغيرات:
    # تعريف متغيرات name = "Khalil" age = 25 print(f"مرحبًا، {name}! عمرك {age} سنة.")
  • الشروط:
    # جملة شرطية age = 18 if age >= 18: print("أنت بالغ!") else: print("أنت قاصر!")
  • الحلقات:
    # حلقة for for i in range(5): print(f"الرقم: {i}") # حلقة while count = 0 while count < 5: print(f"العد: {count}") count += 1

الدوال

الدوال تتيح إعادة استخدام الكود. مثال:

def greet(name): return f"مرحبًا، {name}!" print(greet("Khalil"))

القوائم (Lists)

القوائم تُستخدم لتخزين مجموعة من البيانات:

# إنشاء قائمة fruits = ["تفاح", "موز", "برتقال"] print(fruits[0]) # طباعة: تفاح fruits.append("عنب") # إضافة عنصر print(fruits) # طباعة: ['تفاح', 'موز', 'برتقال', 'عنب']

مشاريع عملية

جرب هذه المشاريع البسيطة لتطبيق مهاراتك:

حاسبة بسيطة

اكتب برنامجًا يطلب من المستخدم إدخال رقمين وعملية حسابية:

num1 = float(input("أدخل الرقم الأول: ")) op = input("أدخل العملية (+, -, *, /): ") num2 = float(input("أدخل الرقم الثاني: ")) if op == "+": print(num1 + num2) elif op == "-": print(num1 - num2) elif op == "*": print(num1 * num2) elif op == "/": print(num1 / num2) else: print("عملية غير صالحة!")

لعبة تخمين الأرقام

برنامج يختار رقمًا عشوائيًا ويطلب من المستخدم تخمينه:

import random secret_number = random.randint(1, 100) guess = 0 while guess != secret_number: guess = int(input("خمن الرقم (1-100): ")) if guess < secret_number: print("الرقم أكبر!") elif guess > secret_number: print("الرقم أصغر!") print("مبروك! لقد خمنت الرقم!")
